# Chat Room App

This is a basic realtime chat application with following features:  
> Login and Register.  
> Authorized API access.  
> Authorized Websocket channels.  
> Realtime Online users status.  
> Realtime messaging.  
> Server and client sync sessions.  

# Resources:
